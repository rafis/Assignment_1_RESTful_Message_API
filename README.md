# Assignment 1 - RESTful Message API #

## Build & Run ##

```sh
$ cd Assignment_1_RESTful_Message_API
$ sbt
> jetty:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.

## Test ##

To run tests:

```sh
$ sbt test
```
